The Solid Share API, allows sites to share links and text to other apps via the system provided share picker.

Sorry! This browser doesn't support the Solid Share API, which required for this demo. 

const shareOpts = {
  title: 'Melvin Carvalho Home Page',
  text: 'Check out the home page of Melvin Carvalho.',
  url: 'https://melvincarvalho.com/',
};
navigator.share(shareOpts);
